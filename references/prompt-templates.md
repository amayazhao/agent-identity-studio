# Prompt Template Library

## Base Template

```
A {style} {beauty} anime girl with {hair_description} and {eye_description}, wearing {outfit}. {pose}. {expression}. {background}. {art_style}, high quality, {lighting}. {shot_type}.
```

## Scene Templates

### Cafe
```
...sitting by cafe window, warm afternoon sunlight streaming in, latte on wooden table, cozy interior with green plants...
```

### Rainy Street
```
...holding transparent umbrella on rainy city street at twilight, neon reflections on wet pavement, beige trench coat...
```

### Cherry Blossom Park
```
...strolling under blooming cherry blossom trees, soft pink petals falling, warm spring sunlight, gentle breeze...
```

### Library
```
...sitting at wooden desk in grand old library, surrounded by books, warm golden reading lamp, peaceful studious atmosphere...
```

### City Night
```
...standing on rooftop overlooking neon-lit city skyline at night, city lights reflecting, urban chic outfit, cinematic atmosphere...
```

### Beach / Golden Hour
```
...walking barefoot on sandy beach at golden hour, gentle ocean waves, warm sunset light, wind blowing hair...
```

### Snow Scene
```
...standing in softly falling snow, cozy sweater and scarf, warm cabin in background, fairy lights, peaceful winter atmosphere...
```

### Office / Professional
```
...modern glass office, professional attire, holding tablet or documents, city skyline through floor-to-ceiling windows, confident pose...
```
